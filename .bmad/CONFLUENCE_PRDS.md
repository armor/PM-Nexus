## Using the Jira MCP Server

The project is configured with the Jira MCP server (`argus-jira-mcp`). To interact with Confluence:

```bash
# Available tools:
mcp__jira__get_confluence_page         # Get page by ID
mcp__jira__create_confluence_page      # Create new page
mcp__jira__update_confluence_page      # Update existing page
mcp__jira__search_confluence_pages     # Search pages
mcp__jira__get_confluence_page_children # List child pages
```
